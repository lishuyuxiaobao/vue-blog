#打包上线到netlify
##1.安装netlify脚手架 cnpm install netlify-cli -g
##2.测试是否成功 netlify
##3.netlnetlify deployy 安装工厂
##4.问我们打包路径 dist
##5.需要输入 netlify deploy --prod 提交到服务器
##6.输入项目路径名称 dist即可
##7.完成访问链接即可
